Pipeline Stages

  1. Quality Control - FastQC analysis of input reads
  2. Reference Indexing - Prepare reference genome for alignment
  3. Read Alignment - Map reads to reference using BWA-MEM
  4. Variant Calling - Call variants using SAMtools/BCFtools
  5. Annotation - Annotate variants (optional)
  6. Report Generation - Create comprehensive HTML report
